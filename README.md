# FontStory

一个字体展示和管理平台。

## 功能特性

- 字体预览和展示
- 字体管理功能
- 响应式设计
- 用户友好的界面

## 项目结构

```
FontStory/
├── admin/          # 管理后台
├── css/            # 样式文件
├── js/             # JavaScript文件
├── ui_screenshots/ # 界面截图
├── favicon.ico     # 网站图标
└── index.php       # 主页面
```
